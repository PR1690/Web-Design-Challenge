Summary: Latitude vs. X

The goal behind this project was to analyze weather change, as you get closer to the equator. For anlaysis, First pulled data from the OpenWeatherMap API and assemble a dataset on over more than 500 cities.

After making suceesful API calls and assemble a data set, we used Matplotlib to plot a graph between weather vs latitude. Other factors were consider durting anlysis like temperature, humindity, cloudiness, and wind-speed. This website provides the source data and visualizations as part of the analysis, on addition detail explanations on trends and correlations.
